# WeChat-Mini-Program
**项目说明**
一款商城小程序项目
### 这是一款仿蘑菇街的商城项目，也是我做的第一个小程序项目，其中仍有不足，待以后技术提高了，再回来完善！
> 主要实现功能有：轮播图、商品展示；切换商品分类；tabControl吸顶效果；上拉加载更多；回到顶部；购物车订单的增添；垂直控制栏商品展示等。

**目录结构：**

- assets——存放项目图片
- components——存放项目公共组件
- pages——存放项目页面相关文件
- service——存放网络请求相关文件
- utils——时间戳转换文件
